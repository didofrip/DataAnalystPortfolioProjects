## 📂 Projects Overview

### 🐍 [Project 5: E-commerce Sales Analysis (Python)](./Project%205%20-%20Python)
**Goal:** Analyze large dataset of electronics sales to identify trends and optimize advertising timing.
* **Tech Stack:** Python (Pandas, Matplotlib, Seaborn, Glob).
* **Key Insights:** Determined optimal ad times (11 AM & 7 PM) and identified product bundles for cross-selling.
* [View Notebook](./Project%205%20-%20Python/Sales%20Product%20Data.ipynb)

### 📈 [Project 4: Advanced Excel Portfolio (4 Modules)](./Project%204%20-%20Excel)
**Goal:** A collection of projects solving distinct business problems using Basic to Advanced Excel capabilities.
* **Tech Stack:** Solver, Scenario Manager, Power Query, Pivot Tables, Advanced Formulas.
* **Key Modules:**
    * **Supply Chain Optimization:** Used **Solver** & Linear Programming to minimize logistics costs under constraints.
    * **Call Center Dashboard:** Built an interactive dashboard using **Sunburst Charts** and Slicers to track agent KPIs.
    * **Retail Sales Analysis:** Conducted Exploratory Data Analysis (EDA) using **Pivot Tables** to identify top-selling categories.
    * **HR Payroll Analysis:** Automated data cleaning and mapping using complex functions (**XLOOKUP, Nested IFs**).

* **Key Achievement:** Transformed raw CSVs into a structured Fact/Dimension model and visualized KPIs like Revenue, Delivery Time, and Customer Satisfaction.
